# Código de conduta

## Compromisso

Como mantenedores e colaboradores, nos comprometemos a tornar a participação no
Dados Financeiros Abertos uma experiência livre de assédio para todas as pessoas, independente
de idade, tamanho, deficiência, etnia, identidade/expressão de gênero,
experiência, nacionalidade, aparência, raça, religião ou orientação sexual.

## Padrão

Comportamentos que contribuem para um ambiente positivo:

- Usar linguagem acolhedora e inclusiva.
- Respeitar pontos de vista e experiências diferentes.
- Aceitar crítica construtiva com graça.
- Focar no que é melhor para a comunidade.

Comportamentos inaceitáveis:

- Uso de linguagem ou imagens sexualizadas, assédio, trolling, ataques pessoais.
- Publicar informação privada de outrem (endereço, email, etc.) sem permissão.
- Qualquer conduta que, razoavelmente, seria considerada inapropriada em um
  ambiente profissional.

## Aplicação

Casos podem ser reportados em privado abrindo uma
[GitHub Discussion privada](https://github.com/robertoecf/findata-br/discussions)
ou direto para [@robertoecf](https://github.com/robertoecf).

Todas as reclamações serão revisadas e investigadas.

## Base

Este código de conduta é adaptado do [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/).
