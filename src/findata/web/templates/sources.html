Rotas rápidas

Exemplos suficientes para começar. O contrato completo fica no OpenAPI.

Macro

BCB e IPEA

GET /bcb/series/name/selic?n=10
GET /ipea/series/BM12_TJOVER12?top=12

Mercado

B3, CVM e ANBIMA

GET /b3/quote/PETR4
GET /cvm/companies/search?q=Petrobras
GET /anbima/ima?index=IMA-B

Dados públicos

Open Finance, IBGE e Tesouro

GET /openfinance/participants?role=DADOS
GET /ibge/ipca/groups
GET /tesouro/bonds/search?q=IPCA

CLI

Mesma fonte no terminal

findata bcb get selic -n 10
findata registry lookup PETR4
findata cvm search Petrobras

Notas de acesso

Regras que não devem ficar escondidas no README.

BD
Base dos DadosGratuito com login para BigQuery

Use `FINDATA_BD_BILLING_PROJECT_ID` para consultas SQL locais. BD Pro permanece marcado como pago.

Ver docs/SOURCES_AND_ENDPOINTS.md
AN
ANBIMAArquivos públicos, não Sensedia

O módulo atual consome XLS/CSV/TXT públicos de `www.anbima.com.br/informacoes/*`.

Ver docs/SOURCES_WITH_AUTH.md
YF
Yahoo FinanceExperimental e não oficial

Prefira fontes oficiais para produção. O endpoint existe para exploração e compatibilidade.
